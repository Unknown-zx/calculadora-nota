# Calculadora de Notas

Programa de consola en Python para gestionar y evaluar las notas de un estudiante.

## Funcionalidades

- Ingresar notas con validación de rango. (0-100)
- Calcular el promedio de todas las notas ingresadas.
- Mostrar la nota más alta y más baja. (requiere mínimo 2 notas)
- Verificar si el estudiante aprobó. (promedio ≥ 70)
- Menú interactivo con manejo de opciones inválidas.

## Tecnologías

- Python 3
- Módulo `time` para mejorar la experiencia de usuario.

## Cómo ejecutarlo

python calculadora_notas.py

## Aprendizajes aplicados

- Funciones con `return` para evitar código duplicado.
- Validación de datos. (rangos, listas vacías, mínimos requeridos)
- Uso de `global` para controlar el flujo del programa.
- Buenas prácticas: separar lógica de cálculo de lógica de presentación.
